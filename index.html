מדיניות פרטיות — מיזם המיליון

אתר זה אוסף מידע מינימלי בלבד לצורך ניהול המיזם:

• ספירת קריאות — כל לחיצה על הכפתור נרשמת עם מזהה אקראי אנונימי שנוצר במכשיר שלכם. לא נשמר שום מידע מזהה אישי כגון שם, כתובת אימייל או IP.

• אחסון נתונים — המזהה האנונימי והספירה נשמרים בשירות Supabase (ספק אחסון נתונים מאובטח). אין שמירה של מידע אישי מזהה.

• תזכורות — אם בחרתם לקבל תזכורות, נעשה שימוש בשירות OneSignal לצורך שליחת התראות דרך הדפדפן. OneSignal מקבל מזהה טכני של המכשיר בלבד, ללא פרטים אישיים.

• עוגיות — האתר אינו משתמש בעוגיות (cookies) לצורכי פרסום או מעקב. נעשה שימוש ב-localStorage בדפדפן לשמירת ההעדפות שלכם באופן מקומי בלבד.

• מנועי חיפוש — האתר מאנדקס על ידי Google לצורך הופעה בתוצאות חיפוש. Google עשויה לאסוף מידע טכני בעת סריקת האתר בהתאם למדיניות הפרטיות שלה.

• שיתוף מידע — לא מועבר שום מידע אישי לצדדים שלישיים למטרות מסחריות. השירותים החיצוניים משמשים לתפעול המיזם בלבד.

• לינקים אישיים — אם בחרתם ליצור לינק אישי, השם או הכינוי שבחרתם נשמר בשרת לצורך מעקב אחר מספר הקריאות שהגיעו דרך הלינק שלכם. הנתונים מוגנים בקוד גישה אישי שנוצר בעת יצירת הלינק. לא נשמר שום מידע מזהה אישי נוסף.

Privacy Policy — The Million Project

This site collects minimal information only for the purpose of managing the project:

• Reading count — Every click on the button is recorded with an anonymous random identifier created on your device. No personally identifiable information such as name, email address or IP is stored.

• Data storage — The anonymous identifier and count are stored in Supabase (a secure data storage provider). No personally identifiable information is stored.

• Reminders — If you choose to receive reminders, OneSignal is used to send browser notifications. OneSignal only receives a technical device identifier, without personal details.

• Cookies — This site does not use cookies for advertising or tracking. localStorage is used to save your preferences locally on your device only.

• Search engines — The site is indexed by Google to appear in search results. Google may collect technical information when crawling the site.

• Data sharing — No personal information is transferred to third parties for commercial purposes. External services are used solely to operate the project.

• Personal links — If you choose to create a personal link, the name or nickname you chose is stored on the server to track the number of readings that came through your link. The data is protected by a personal access code created when the link is created. No additional personally identifiable information is stored.

Politique de confidentialité — Le Projet du Million

Ce site collecte un minimum d'informations uniquement pour la gestion du projet:

• Comptage des lectures — Chaque clic sur le bouton est enregistré avec un identifiant anonyme aléatoire créé sur votre appareil. Aucune information personnelle identifiable telle que nom, adresse e-mail ou IP n'est stockée.

• Stockage des données — L'identifiant anonyme et le comptage sont stockés dans Supabase (fournisseur de stockage de données sécurisé). Aucune information personnelle identifiable n'est stockée.

• Rappels — Si vous choisissez de recevoir des rappels, OneSignal est utilisé pour envoyer des notifications via le navigateur. OneSignal ne reçoit qu'un identifiant technique de l'appareil, sans données personnelles.

• Cookies — Ce site n'utilise pas de cookies à des fins publicitaires ou de suivi. Le localStorage est utilisé pour sauvegarder vos préférences localement sur votre appareil uniquement.

• Moteurs de recherche — Le site est indexé par Google pour apparaître dans les résultats de recherche. Google peut collecter des informations techniques lors de l'exploration du site.

• Partage de données — Aucune information personnelle n'est transmise à des tiers à des fins commerciales. Les services externes sont utilisés uniquement pour faire fonctionner le projet.

• Liens personnels — Si vous choisissez de créer un lien personnel, le nom ou le surnom que vous avez choisi est stocké sur le serveur pour suivre le nombre de lectures venues via votre lien. Les données sont protégées par un code d'accès personnel créé lors de la création du lien. Aucune information personnelle supplémentaire n'est stockée.
